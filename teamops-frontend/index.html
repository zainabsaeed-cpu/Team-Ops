Core features

Built for teams who
ship fast

Real-Time Sync
Socket.io broadcasts every card move and comment to all connected clients in under 50ms.
Kanban Boards
Drag-and-drop cards across custom workflow stages. Position persisted in PostgreSQL.
Role-Based Access
Owner, Admin, Member roles. JWT auth with bcrypt password hashing throughout.
Live Notifications
Get pinged on assignments and @mentions. Badge counts and a full notification history page.
Activity Feed
Every action logged and surfaced live in a sidebar — who moved what, when, with context.
PostgreSQL Backend
Relational storage with JOINs, FK constraints, ordered positions, and indexed queries.
